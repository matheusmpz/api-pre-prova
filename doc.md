- Criar projeto
laravel new example-app

- Abrir projeto
cd example-app
npm install && npm run build

- Arquivo de Rotas
php artisan install:api

- Rodar API
php artisan serve

- Arquivo Model
php artisan make:model NOME

- Criação de migração
php artisan make:migration (ex: create_products_table)

- Rodar o migration
php artisan migrate

- Verifica as relações
php artisan model:show MODEL
